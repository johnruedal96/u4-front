# 🏟️ U4 Front - UEFA Match Manager

Frontend construido con **React + Vite + TypeScript** y **TailwindCSS** para gestionar partidos de la UEFA. Puedes crear, editar, eliminar, buscar y cargar múltiples partidos desde archivos CSV.

## 🚀 Características

- 📋 Listado de partidos en tabla con filtros
- 🔎 Filtro de texto
- 📁 Carga masiva de partidos desde CSV
- 🖊️ Formulario para crear y editar partidos
- ⚙️ Integración con backend (Spring Boot)

## ⚙️ Variables de entorno

Crea un archivo `.env` en la raíz:

```env
VITE_API_URL=http://localhost:8100
```

# Instala las dependencias

npm install

# Corre en modo desarrollo

npm run dev

# Compila para producción

npm run build

# Sirve el build localmente

npm run preview
